🤖 Simple GenAI App using LangChain + Ollama + Streamlit

A simple Generative AI web application built using **LangChain**, **Ollama (Gemma 2B model)**, and **Streamlit**.  
This app allows users to ask questions and receive AI-generated responses through a clean and interactive web interface.

---

## 🚀 Features
- 🔗 Built with **LangChain**
- 🤖 Uses **Ollama (Gemma 2B model)**
- 🌐 Interactive UI using **Streamlit**
- 🔐 Environment variable support using `.env`
- ⚡ Simple **Prompt → LLM → Output Parser** chain

---

## 🛠️ Tech Stack
- Python
- LangChain
- Ollama
- Streamlit
- python-dotenv

---

## 📋 Prerequisites
Make sure you have the following installed:

- ✅ Anaconda / Miniconda
- ✅ Python 3.9+
- ✅ Ollama
- ✅ Gemma 2B model pulled

---

## 🧠 How It Works
- User enters a question in the **Streamlit UI**
- LangChain formats the prompt
- Ollama’s **Gemma 2B** model processes the query
- Output is parsed using `StrOutputParser`
- Answer is displayed instantly
